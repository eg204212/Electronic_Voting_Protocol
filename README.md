# Secure Electronic Voting System

This project is a Java-based secure electronic voting system that ensures:

* ✅ Authentication (only eligible voters can vote)
* 🕵️ Anonymity (votes are unlinkable to voters)
* 🧾 Integrity (votes cannot be tampered with)
* 🔐 Confidentiality (votes are encrypted)

## Features

* Digital certificate generation and verification
* One-time voting token issuance
* Encrypted vote casting
* Secure vote storage and result counting
